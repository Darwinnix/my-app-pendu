Bienvenue dans My app: Pendu

Ce jeu est un jeu de pendu classique. Le but est de trouvé un mot en proposant des lettres.

  • Tapez la touche de la lettre souhaitée. Si la lettre en question est contenue dans le mot mystère une ou plusieurs fois, elle s'affichera automatiquement.
  • Chaque fois qu'une lettre est contenue dans le mot, le nombre d'essais ne change pas.
  • Si la lettre choisie n'est pas contenue dans le mot, le nombre d'essais s'incrémente de 1 (et le dessin du pendu se complète).
  • La partie est gagnée si le mot est totalement découvert avant que le dessin du pendu soit complet (soit 10 essais).
  • La partie est perdue si le dessin est complété avant de découvrir le mot au complet.
  • Attention, les lettres e, é et è sont différentes dans cette version du pendu.
